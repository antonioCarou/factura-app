# 🧾 Factura App

Aplicación local de facturación desarrollada con React, Node.js, SQLite y generación de PDFs.

## 🔧 Instalación

```bash
cd backend
npm install
cd ../frontend
npm install
# Terminal 1 (backend)
cd backend
npm run dev

# Terminal 2 (frontend)
cd frontend
npm run dev
```

## ▶️ Ejecución

```bash
# Terminal 1 (backend)
cd backend
npm run dev

# Terminal 2 (frontend)
cd frontend
npm run dev
```

## 🧠 Funciones
Generación de facturas en PDF
Histórico de productos y precios
Autocompletado de clientes y productos
Listado y descarga de facturas
